

# React To-Do List App
Features:

# Task List Display:

Show a list of tasks with their status (completed or not).
Include the task name and a checkbox to mark tasks as complete.

# Add New Task:
Allow users to add new tasks to the list.
Input field for task name and an "Add" button.

# Remove Task:
Allow users to remove tasks from the list (for example, a delete button next to each task).

# Task Filtering:
Add a feature to filter tasks based on their status (all, completed, or active).

# Task Count:
Display the total number of tasks and the number of completed tasks.
